# A2A Protocol Branding

## Logo

The A2A Protocol logo represents two agents connecting:

- **Two circles** = Two agents
- **Connection line** = Communication between them
- **Center node** = The protocol bridging them

### Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Purple | `#4f46e5` | Main brand color |
| Secondary Violet | `#7c3aed` | Gradient end |
| White | `#ffffff` | Text, icons |
| Dark Background | `#1a1a2e` | Dark mode |

### Typography

- **Primary Font**: System fonts (Arial, -apple-system, Roboto)
- **Bold** for headings
- **Regular** for body text

## Files

### SVG (vector - preferred)
- `logo.svg` - Main logo (200x200)
- `favicon.svg` - Favicon (32x32)

### Usage

**Light backgrounds:**
```html
<img src="logo.svg" alt="A2A Protocol" height="60">
```

**Dark backgrounds:**
The logo works on both (white elements).

**Favicon:**
```html
<link rel="icon" type="image/svg+xml" href="favicon.svg">
```
